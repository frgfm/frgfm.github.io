## Wildfire early detection system
![](https://github.com/frgfm/PyroNear/raw/master/docs/source/_static/img/pyronear-logo-dark.png)

The [PyroNear](https://github.com/frgfm/PyroNear) project aims at providing a state-of-the-art wildfire detection system that can easily be deployed on low-cost hardware. Instead of high-end equipment, we focus on designing a high-performing detection model to be run on a Raspberry Pi device.

Volunteers from [Data for Good](https://dataforgood.fr/) joined the project to prevent wildfires from spreading in areas without the means to deploy either human resources or high-end automated solutions.
